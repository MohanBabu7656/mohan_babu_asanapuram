# **App Name**: Mohan_Babu_Asanapuram

## Core Features:

- Interactive Project Gallery: Showcase creative projects with image and video uploads, enhanced by a visually engaging interactive design with hover effects.
- Detailed Project Descriptions: Allow for in-depth descriptions of each project, providing context and insights into the development process.
- About Me Section: Introduce yourself with a professional 'About Me' section detailing your background, skills, and expertise.
- Contact Form: Enable direct communication through an integrated contact form for inquiries and collaborations.
- Animated Typing Hero Text: Feature an animated typing effect for titles in the hero section.
- Dynamic Content Tool: Utilize AI to analyze skills and experience to tailor portfolio content dynamically, ensuring relevance and appeal to visitors.

## Style Guidelines:

- Primary color: Deep Charcoal (#333333) for a modern, dark background.
- Background color: Near-black (#1A1A1A) with slight desaturation to enhance contrast and depth.
- Accent color: Electric Purple (#800080) to add a vibrant, premium feel, inspired by modern technology aesthetics.
- Headline font: 'Space Grotesk' sans-serif for headlines and short amounts of body text.
- Body font: 'Inter' sans-serif font for longer passages of body text.
- Use a set of minimalist icons for skills and technologies. Icons should complement the electric purple and neon blue accent colors.
- Implement a fully responsive design with a sticky navigation bar and smooth scrolling to ensure optimal viewing on all devices.
- Employ smooth transitions and subtle Framer Motion animations to enhance user experience.
